# Conditionals, Loops & Operators

->An operator is a symbol that tells the engine to perform a specific action on one or more values.

## The 4 fundamental operator
    1.Arithmetic – change numbers
    2.Assignment – store results
    3.Comparison – produce true/false
    4.Logical – combine conditions

## data conversion 
-> We know if we try to compare string to number js convert string into number 

**javaScript** 

    <!-- first case  -->
    a="100";
    b= number(a); 
    console.log(b) // output => 100 as a number

    <!-- second case  -->
    a="100abc";
    b= number(a); 
    console.log(b) // output => NaN => Not a number.

    <!-- third case  -->
    console.log(0/0) // output => NaN => Not a number.
